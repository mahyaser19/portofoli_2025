Getting Started with Flask

Flask is a lightweight and flexible Python web framework that provides the tools, libraries, and technologies needed to build a web application. This web application can be some web pages, a blog, a wiki or go as big as a web-based calendar application or a commercial website.

Key Features of Flask:

  • Lightweight: Flask is a micro-framework that keeps the core simple but extensible.
  • Flexible: You have the freedom to choose how you want to implement things.
  • Easy to Learn: Flask has a gentle learning curve and excellent documentation.
  • Extensible: Flask has a rich ecosystem of extensions.
Basic Flask App Structure 
from flask import Flask

app = Flask(__name__)

@app.route('/')
def home():
    return 'Hello, World!'

if __name__ == '__main__':
    app.run(debug=True)
